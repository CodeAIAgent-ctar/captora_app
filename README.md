# Captora Website

The official community portal for **Captora** — a Windows screen text extraction tool that lets you instantly copy text from anywhere on your screen with `Win + Q`.

## Tech stack

- Pure static HTML/CSS/JS — no build step required
- Hosted on Netlify

## Run locally

Open `index.html` directly in a browser, or serve it with any static file server:

```bash
npx serve .
```
